# Current PE Labs Aesthetic Archive

This archive preserves the pre-overhaul PE Labs visual system before the Humanist Compute Atelier rebuild.

## Snapshot

- Pre-overhaul commit: `eedfae7`
- Live app structure: Next.js App Router
- Primary visual system: cream paper canvas, moss and sage accents, serif-led headings, rounded product panels
- Public data boundary: synthetic organisations only

## Files

- `theme-notes.md` records the existing colour, type and layout approach.
- `restoration-notes.md` explains how to restore this aesthetic from Git.
- `component-map.md` lists the main routes and components affected by the overhaul.
- `css-or-theme-snapshot.css` is a direct snapshot of `src/app/globals.css` before replacement.

## Overhaul Intent

The replacement should keep the existing product structure but make the workflow more connected:

1. Company intelligence
2. Investment screening
3. Relationship drafting

Selected company context should flow through all three modules without introducing live client data or hidden scoring logic.
