# Neo-Renaissance Protocol Note

- Date: 2026-06-26
- Register: workshop and laboratory
- Source: Charlie's Neo-Renaissance aesthetic protocol and PE Labs shot-list entry

PE Labs is treated as a private-equity field laboratory: company intelligence as a portfolio field, screening as an inspectable rubric, and outreach as a drafting desk.

The refresh uses the existing generated hero tableau as a painterly base plate. No new image assets were generated for this pass. The product layer applies deterministic CSS treatment over the app: paper grain, measurement ticks, dot matrices, ledger rules, restrained verdigris accents, compact cards and responsive safe zones.

The aesthetic is deliberately subordinate to the workflow. Routes, data logic, synthetic-demo labelling, screening scores, drafting provider boundaries and public claims are preserved.

The previous visual direction remains archived in `docs/archive/current-pe-labs-aesthetic/`.
