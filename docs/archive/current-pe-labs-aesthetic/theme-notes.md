# Theme Notes

## Summary

The pre-overhaul PE Labs aesthetic is a clean private-market product demo with a cream and green palette, serif headings, rounded cards and light dashboard structure.

## Tokens

- Main canvas: parchment and paper tones
- Text: deep green ink with muted secondary copy
- Accents: moss, sage, clay and gold
- Surfaces: translucent paper cards with thin green hairlines
- Radius: mostly 11px to 18px
- Shadow: soft green-grey paper shadow

## Typography

- Display: `Newsreader`
- Body: `Manrope`
- Mono labels: `JetBrains Mono`

## Layout

- Sticky top navigation with segmented module links
- Large home hero with abstract landscape graphic
- Three-module card grid
- Company universe as card grid
- Screening as desktop ledger table with mobile cards
- Outreach as two-column drafting form and output panel

## Strengths To Preserve

- Clear module IA
- Synthetic-demo honesty
- Deterministic screening language
- Company monograms
- Concise company detail pages
- "Write with a reason" outreach concept

## Reasons For Replacement

- Module state is not yet visibly shared across Intelligence, Screening and Outreach.
- Visual language is pleasant but not yet a distinctive Humanist Compute Atelier system.
- Screening traceability is present in detail pages but not prominent enough in the module flow.
- Mobile needs to feel like a deliberate field book rather than a compressed desktop surface.
